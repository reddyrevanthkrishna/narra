# Narra

Narra is a premium resale marketplace that enables buyers to discover trusted sellers, compare prices, and purchase premium products with confidence.

## Technology Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- Python
- FastAPI
- SQLAlchemy
- Alembic
- Pydantic

### Database
- PostgreSQL

### DevOps
- Docker
- GitHub Actions
- Nginx
- AWS
